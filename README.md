Анализ данных о видеоиграх

Проект по исследованию датасета, содержащего информацию о 146 популярных видеоиграх: от классики (Super Mario Bros., 1985) до современных хитов (Elden Ring, 2022). В рамках проекта проведён разведочный анализ данных (EDA) в Python и создана интерактивная аналитика в Yandex DataLens.

Описание проекта

Цель проекта — выявить закономерности между:



Жанром игры и её рейтингом



Годом выпуска и популярностью



Средним рейтингом и количеством отзывов



Количеством играющих и отложенных игр



Анализ помогает ответить на вопросы:



Какие жанры и разработчики получают самые высокие оценки?



Как менялись игровые предпочтения с течением времени?



Существует ли связь между количеством отзывов и рейтингом?



Какие игры чаще попадают в списки желаемого и отложенных?



Исходные данные

Датасет: games.csv (146 записей)



Поля:



Title — название игры



Release Date — дата выпуска



Team — разработчики (список)



Rating — средняя оценка (1-5)



Times Listed — сколько раз игра добавлена в списки



Number of Reviews — количество отзывов



Genres — список жанров (например, \['Adventure', 'RPG'])



Summary — краткое описание



Reviews — тексты отзывов



Plays — сколько раз сыграли



Playing — сколько играют сейчас



Backlogs — в списке "отложенных"



Wishlist — в списке желаемого



Датасет включает как классику игровой индустрии (Super Mario 64, Chrono Trigger), так и современные хиты (Elden Ring, Baldur's Gate 3).



Анализ в Python

В файле games\_analysis.ipynb выполнен полный EDA с использованием pandas, seaborn и matplotlib.



Выполненные шаги:

Загрузка и осмотр структуры данных



Обработка пропусков в числовых колонках



Преобразование списков (Genres, Team) из строк в Python-объекты



Конвертация дат в формат datetime



Обработка числовых значений с 'K' (тысячи)



Визуализации:

Распределение рейтингов (гистограмма)



Топ-10 игр по количеству играющих (столбчатая диаграмма)



Динамика выпуска игр по годам (линейный график)



Boxplot рейтинга по жанрам (боксплот)



Тепловая карта корреляций между метриками





Ключевые выводы:

Средний рейтинг игр: 4.02 — игроки склонны оценивать высококачественные игры положительно



Самые популярные жанры: Adventure, RPG, Indie



Наиболее высокооценённые игры: Disco Elysium (4.6), Elden Ring (4.5), Bloodborne (4.5)



Сильная корреляция (0.85) между Plays и Backlogs — популярные игры часто откладывают "на потом"



Современные игры (2020-2023) имеют более активное комьюнити (Playing), но классика имеет более стабильные рейтинги
